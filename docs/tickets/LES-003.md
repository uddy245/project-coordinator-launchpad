# LES-003: Lesson page shell (video / workbook / quiz tabs)

**Milestone:** M3 Lesson Delivery
**Dependencies:** LES-002
**Estimated session length:** medium (30–90 min)
**Status:** not started

## Context

`/lessons/[slug]` is the central learner surface. This ticket sets up the shell: header with lesson title + breadcrumb, three tabs (Video / Workbook / Quiz) where the active tab is driven by a `?tab=` search param so links can deep-link. Tab content comes in LES-004 / LES-005 / LES-007.

## Files to create

- `src/app/(app)/lessons/[slug]/page.tsx` — server component, fetches lesson, renders header + active tab
- `src/components/lessons/lesson-header.tsx`
- `src/components/lessons/tabs.tsx` — client component reading/writing `?tab=`
- `tests/e2e/lesson-tabs.spec.ts`

## Acceptance criteria

- [ ] Visiting `/lessons/raid-logs` renders the lesson header with title "RAID Logs" and breadcrumb "Dashboard → Lesson 20: RAID Logs"
- [ ] Tabs row shows Video / Workbook / Quiz; default active is Video
- [ ] Clicking a tab updates the URL (`?tab=video|workbook|quiz`) without a full reload
- [ ] Direct navigation to `/lessons/raid-logs?tab=quiz` opens the Quiz tab
- [ ] Unknown slug → 404 via `notFound()`
